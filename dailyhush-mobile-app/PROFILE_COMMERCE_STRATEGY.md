## ğŸ› ï¸ PART 7: TECHNICAL IMPLEMENTATION

### **New Database Tables Needed:**

**1. Products Table**
```sql
CREATE TABLE products (
  id UUID PRIMARY KEY,
  name TEXT,
  description TEXT,
  price_usd DECIMAL,
  product_type TEXT, -- 'digital' | 'physical' | 'service'
  loop_types TEXT[], -- ['sleep-loop', 'decision-loop']
  image_url TEXT,
  affiliate_link TEXT,
  is_premium_only BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP DEFAULT NOW()
);
```

**2. Product Recommendations Table**
```sql
CREATE TABLE product_recommendations (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES user_profiles(user_id),
  product_id UUID REFERENCES products(id),
  recommendation_reason TEXT, -- "Based on your Sleep Loop patterns"
  shown_at TIMESTAMP,
  clicked_at TIMESTAMP,
  purchased_at TIMESTAMP,
  created_at TIMESTAMP DEFAULT NOW()
);
```

**3. Purchases Table**
```sql
CREATE TABLE purchases (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES user_profiles(user_id),
  product_id UUID REFERENCES products(id),
  amount_usd DECIMAL,
  purchase_type TEXT, -- 'stripe' | 'affiliate' | 'bundle'
  status TEXT, -- 'pending' | 'completed' | 'refunded'
  created_at TIMESTAMP DEFAULT NOW()
);
```

**4. Brand Collaborations Table**
```sql
CREATE TABLE collaborations (
  id UUID PRIMARY KEY,
  brand_name TEXT,
  collaboration_type TEXT, -- 'affiliate' | 'sponsored' | 'co-branded'
  loop_types TEXT[],
  active BOOLEAN DEFAULT TRUE,
  commission_rate DECIMAL,
  created_at TIMESTAMP DEFAULT NOW()
);
```

### **New API Endpoints:**

- Full product details page

**POST /api/purchases**
- Creates purchase record
